# Changelog

## [0.4.0] - 2026-05-16

### Added
- Pomodoro timer with status bar countdown and phase notifications
- Break reminder after configurable hours of continuous coding
- Session notes and tags (multi-select) saved to last session
- Export JSON via Save dialog
- Tag filter chips in dashboard overview
- Recent sessions table in project detail (date, duration, tags, notes, Pomodoro count)
- Pomodoro badge on project cards
- Daily goal achievement notification (once per day)

### Improved
- CSV export: added `duration_h`, `tags`, `notes` columns

## [0.3.0]

### Added
- Archive / unarchive projects
- Config tab with per-project rate, currency, daily/weekly goals
- "Configure" button in project detail view

### Fixed
- Language percentage calculation (was always 100%)
- Goal progress bars now show in project cards

## [0.2.0]

### Added
- Cost tracking with hourly rate per project
- Daily and weekly goal tracking with progress bars
- Streak counter
- Language/extension breakdown per project and globally
- AI usage tab (Claude Code log parsing)

## [0.1.0]

### Added
- Automatic time tracking per project
- Idle detection
- Dashboard with project overview and detail view
- Status bar with active project and elapsed time
